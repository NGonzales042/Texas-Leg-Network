# Texas-Leg-Network
Network analysis of Texas legislators using Python and Tableau.

A data engineering and network analysis project exploring relationships among members of the Texas Legislature. This project transforms publicly available legislative data into a network model that can be visualized in Tableau and expanded with machine learning techniques.

Project Overview
The goal of this project is to model how legislators are connected through committee assignments, districts, legislative service, and other shared characteristics. Rather than viewing legislators individually, this project examines the legislature as a network.
